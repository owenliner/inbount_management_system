# 商品入库系统

参考material_cos实现的入库管理系统代码，实现一套商品管理系统（java+_vue实现的代码），我需要用python+react重构系统；另外还有两个系统（商品查询系统和商品发货系统）依赖入库管理系统。

# 架构设计参考

三个系统的关联性分析
这三个系统存在强业务关联，它们共同构成了一个完整的商品仓储管理系统（WMS） 的核心模块：
商品入库 → 库存变动 → 商品发货
              ↑
         商品查询（贯穿全程）

关联关系详解
系统核心职责与其他系统的关系入库管理商品进入仓库、库存增加为查询系统提供入库记录；影响可发货库存发货管理商品出库、库存减少依赖入库后的库存数据；产生发货记录供查询查询管理信息检索、报表统计聚合入库和发货数据，提供统一查询视图

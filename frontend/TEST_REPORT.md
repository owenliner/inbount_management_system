## 控制台警告（非阻塞性问题）

测试过程中发现以下控制台警告，不影响功能但建议后续优化：

1. **Ant Design 废弃 API 警告**
   - `dropdownRender` 已废弃，建议使用 `popupRender`
   - `InputNumber` 的 `addonAfter` 已废弃，建议使用 `Space.Compact`

2. **React Router Future Flag 警告**
   - 建议添加 `v7_startTransition` flag
   - 建议添加 `v7_relativeSplatPath` flag
